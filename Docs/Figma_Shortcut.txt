*) Show/Hide Guide:
    -> Shift + R
    
*) Command palette
    -> Ctrl + p
    
*) Get the distance between selected element to other element
    -> Alt + Hover Other element
    
*) Select the required child element inside frame
    -> Ctrl + Right Click

*) Select multiple components:
    -> Shift + Right click components

*) Group Selected Element inside new Frame & Auto Layout:
    -> Shift + A
    
*) Group Selected Element inside Group frame:
    -> Ctrl + g

*) Move Selected Element:
    -> Left
    -> Right
    -> Up
    -> Down
    -> Shift + Left
    -> Shift + Right
    -> Shift + Up
    -> Shift + Down

*) Duplicate Element:
    -> Alt + Grab Element

*) Sample color from the figma design playground and use it as a background image of selected element:
    -> i


*) Duplicate variables:
    -> Shift + Enter

*) Create new Figma file:
    -> Ctrl + n

*) Show/Hide Editor UI:
    -> Ctrl + \
